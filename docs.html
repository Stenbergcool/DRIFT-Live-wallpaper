Quick start

Welcome to Drift.

Drift turns your own videos into a live desktop wallpaper on Windows 11. Drop a clip on the timeline, choose how it spreads across your monitors, and it plays quietly behind your icons. There's no account — your videos and settings stay on your PC.

movie

Your videos, live

Use any video files you already have. Drift plays them on the desktop layer — a single clip or a whole sequence.

bolt

Light on resources

Muted by default, runs from the tray, and pauses itself for full-screen apps and on battery.

Installing

info

Requirements: Windows 11, 64-bit. Drift installs for your user only, so it doesn't ask for administrator rights.

  1. 1

    Download the installer

    Grab the latest Drift_x64-setup.exe from the download link. It's a standard Windows installer — no command line needed.

  2. 2

    Run it

    Double-click the installer and follow the prompts. Because Drift is a new app, Windows SmartScreen may show a warning the first time — choose More info → Run anyway to continue. Drift opens and adds an icon to your system tray.

  3. 3

    Set your first wallpaper

    On the Wallpaper tab, drag a video onto the timeline (or click browse files). It starts playing on your desktop straight away.

The interface

The control window has a slim rail on the left with three tabs. Closing the window doesn't quit Drift — it keeps running in the tray.

wallpaper

Wallpaper

The main workspace: your clip timeline, the display layout, and presets all in one place.

grid_view

Saved videos

Your library — videos you've imported into Drift so they're always a click away.

settings

Settings

Audio, power behaviour, and launch-at-startup.

Multi-monitor

Pick how a clip spreads across your displays with the mode switch above the layout area. Each clip can use a different mode.

Every screen

The same video plays on every monitor, decoded once and mirrored to each display.

Span

One video is stretched across your whole monitor arrangement, with each screen showing its slice — great for a single ultra-wide scene.

Custom

Open the visual layout editor with the live video as the backdrop. Drag and resize a box over each monitor to choose exactly which part of the video it shows.

  • • Crops are stored as a centre point and zoom, so they never distort and survive swapping in a different video.
  • • Assign a different video to an individual monitor, or leave it on the shared one. Per-monitor videos always play muted.
  • • Turn a monitor off in the layout and it shows plain black.

Timeline & clips

A wallpaper can be a single looping clip or a sequence. Add clips to the timeline and the wallpaper plays through them in order, then loops back to the start.

repeat How long each clip stays

Loops

Play the clip a set number of whole times, then move on.

Hold

Keep looping the clip until a number of seconds of real time have passed.

Forever

Never advance — for a single clip, or a final one that should loop indefinitely.

animation Transitions

Choose how each clip hands off to the next: a hard cut, or a crossfade with a length you set in milliseconds.

folder Groups

Bundle several clips into a group that repeats as a unit a set number of times before the timeline carries on. Groups can be nested.

Library & presets

video_library

Library

Importing a video copies it into Drift's own folder, so it stays available even if you move or delete the original. Rename entries to keep things tidy; removing one cleans up anything that was using it.

bookmark

Presets

Save the whole composition — your timeline and display layout — as a named preset, then switch between them in a click. Your library, audio and power preferences stay global and aren't part of a preset.

Settings & tray

Audio

Off by default. Turn it on to hear the shared video's own sound and set a volume. Per-monitor videos always stay muted.

Pause on battery

Stops playback when a laptop is unplugged to save power.

Launch at startup

Start Drift quietly in the tray when you sign in to Windows.

expand_circle_down System tray

Drift lives in the tray. Left-click the icon to open the window; right-click for a menu to Open Drift, Pause wallpaper, or Quit. Quitting restores your normal Windows wallpaper.

FAQ

Is there a macOS version?expand_more

Not yet — Drift is built for Windows 11 only. It relies on the Windows desktop window layer to sit behind your icons, which doesn't exist on macOS.

What video formats can I use?expand_more

Drift plays videos through the system's built-in web video engine, so common web-friendly formats work best — an MP4 (H.264) is the safest bet. WebM also works. If a file won't play, re-encoding it to MP4/H.264 usually fixes it.

Will it slow down my games or drain my battery?expand_more

Drift pauses itself when a full-screen app or game is in front and when your laptop is on battery (both on by default), and it's muted unless you turn audio on — so it gives the machine back when you need it.

Where are my videos and settings stored?expand_more

Everything stays local. Imported videos are copied into Drift's folder in your Windows user app-data, and your settings and presets are saved there as plain files. Nothing is sent to a server and there's no account.

Does it need administrator rights?expand_more

No. Drift installs just for your user account, so it doesn't require admin privileges.
